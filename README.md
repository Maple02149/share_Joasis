## Start Project
```
git clone https://github.com/Maple02149/share_Joasis.git
cd share_Joasis
```
add .env in server and open .env file to input the datail
```
cd .\server\
cp .\.env.example .env
code . 
```

At server and project open the web/server
```
yarn dev
```
